# Kit administratif Macao

## Check-list complète

- Kbis récent
- RIB
- Attestation URSSAF
- Attestation fiscale
- Assurance RC Pro
- Statuts SAS 2D COM
- Pouvoir de signature
- Logo Agence Macao
- Présentation PDF Agence Macao
- DC1 prérempli
- DC2 prérempli
- Références clients
- CV / profils équipe
- Modèle de mémoire technique
- Modèle d’offre financière
- Modèle de planning
- Modèle de courrier de candidature

## Usage

- conserver les pièces à jour dans un dossier Drive dédié
- vérifier les dates de validité avant chaque dépôt
- dupliquer le kit pour chaque réponse nécessitant un ajustement spécifique
