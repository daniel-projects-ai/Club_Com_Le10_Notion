Cœur du système — appels d'offres & projets

  • Acheteur / clienttexte

  • Budget estiménombre €
  • Date limitedate
  • Statutliste
  • Résumé IA · Score IAtexte · nombre
  • Métiers · Coworkers suggérés IAliens
← Macao (import Notion + saisie)

2 · Coworkers

Les membres du 10

  • Prénom · Nomtexte
  • Emailemail · clé
  • Bio · Portfolio · Photodivers
  • Métiers / compétenceslien
  • Disponibilitéliste
  • StatutEn attente / Actif
  • RôleCoworker / Macao
← Jotform (inscription)

3 · Métiers / Compétences

Référentiel — pivot du matching

  • Métiertexte · principal
  • Catégorieliste
  • Opportunitéslien
  • Coworkerslien
← Macao (référentiel fixe)

5 · Commentaires

  • Opportunitélien
  • Auteurlien → Coworker

  • Dateauto
← Interface

6 · Contacts / Demandes

Messages externes

  • Nom · Emailtexte · email
  • Type de demandeliste
  • Messagetexte long
  • Statut de traitementliste
← Jotform (contact)

Flux de données

  1. Inscription — le coworker remplit le Jotform → table Coworkers, Macao valide.
  2. Nouvelle opportunité — Macao la saisit → l'IA remplit résumé, métiers, score.
  3. Matching — l'IA suggère les coworkers dont les métiers correspondent.
  4. Interaction — le coworker clique « ça m'intéresse » et commente dans l'interface.
  5. Pilotage — Macao suit les positionnements et fait avancer le statut.
  6. Contact externe — un prospect écrit via le Jotform → table Contacts.

Accès par rôle

🛠️ Macao (admin)

  • Gestion complète des opportunités & statuts
  • Vue des positionnements & validation des inscriptions
  • Traitement des contacts, déclenchement de l'IA

👥 Coworkers

  • Consultation des opportunités visibles
  • « Ça m'intéresse » & commentaires
  • Annuaire des coworkers — lecture seule sur les données Macao
